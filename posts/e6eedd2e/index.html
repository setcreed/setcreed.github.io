前言

Linux操作系统应用越来越广，现在企业大多用的是redhat系的Linux发行版本，

但是随着云计算的发展，ubuntu被有些云计算作为推荐使用的首选操作系统平台。

redhat系列的教程很多，而ubuntu课程很少。

redhat是收费的，而ubuntu是完全开源的免费的。学自苑老师的Ubuntu Server教程

操作系统

Operating System（OS）

  • 管理和控制计算机软硬件的计算机程序
  • 向下控制和使用计算机硬件
  • 向上为其他应用软件提供软件运行环境

各种操作系统的类型

  • Unix
  • Windows
  • LInux
  • DOS
  • Netware
  • OS X
  • Android
  • iOS

我们只要关注Unix、windows、Linux这些操作系统，就涵盖了市面上几乎所有的操作系统，其他的都是它们三者的衍生。

操作系统的用途

服务器操作系统

  • 稳定性是最主要的指标
  • 没有图形界面

客户端操作系统

  • 速度以及丰富的新特性是主要指标
  • 丰富的图形化客户端软件

Linux操作系统

操作系统内核直接与硬件交互

Linux发行版本：

  • Redhat
  • Debian
  • SusLinux
  • ArchLinux
  • Gentoo

我们一般说的Linux通常指的是内核，如果说到Linux的某一个发行版本，那么就包含了内核和很多的应用软件包打包在一起的发行版本。

发行版本（distribution）：https://distrowatch.com/

Ubuntu Linux

  • 基于Debian的Linux发行版本
  • 开源、免费、易用
  • 支持平台：桌面、服务器、loT、云平台、移动设备
  • 支持32位和64位硬件平台
  • 版本支持
    • LTS(Long Term Support) 长期支持版，5年技术支持
    • 非LTS版本支持9个月，单数版本是非LTS
    • 每六个月发布一个新版本（在4月、10月）

官方网址：http://www.ubuntu.com/

ubuntu Linux的衍生版本：

  • Ubuntu
  • Ubuntu Server
  • Kubuntu
  • Xubuntu
  • Lubuntu
  • Ubuntu Kylin

Ubuntu Server与Desktop版本的差别

现在几乎没有差别了

  • Server默认不安装X Windows 桌面环境
  • 10.10及以前版本，server和desktop的内核不同，但现在全部使用了通用内核（generic kernel）
  • 服务器版本和桌面版本使用完全相同的软件仓库
